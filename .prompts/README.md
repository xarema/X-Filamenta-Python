# Pack de prompts (Audit + Nettoyage) — Copilot / Claude / GPT / Junie / Gemini / Grok

Ce ZIP contient **2 prompts** (Audit+Fix, Nettoyage+Docs+.gitignore) pour chacun des outils suivants :
- GitHub Copilot
- Claude
- GPT (ChatGPT / GPT-*) 
- Junie (JetBrains)
- Gemini (Google)
- Grok (xAI)

## Comment utiliser
1) Ouvre ton dépôt localement (ou charge le repo / connecte le projet dans l’outil).
2) Choisis le dossier de l’outil que tu utilises.
3) Copie-colle `01_Audit_Securite_Qualite_Fix.md` puis exécute/valide.
4) Ensuite copie-colle `02_Nettoyage_Docs_Gitignore.md`.

## Important
- Les prompts exigent de **lire et respecter d’abord** les règles / guidelines / policies situées dans `.github/` (et tout autre fichier de règles d’agent si présent).
- Les prompts sont conçus pour produire des résultats **niveau senior**, avec plan, priorisation, diffs/patches, et vérifications (tests/CI) quand c’est possible.
